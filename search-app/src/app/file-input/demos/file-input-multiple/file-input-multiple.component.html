[clearFilesOnAdd]="{{clearFilesOnAdd}}" Existing list of uploaded files will be cleared on adding new files New files will be appended to pre-existing list of files

Note: Default behavior in USWDS is to clear any uploaded files when new files are added. When using the config to append new files, we leave it up to the application to build the user interface for removing files, but do provide two functions - clearFiles() to remove all files and removeFile(File) to remove a single file in the file input component that users may call
